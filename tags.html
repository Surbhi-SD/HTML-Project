What are Tags

HTML tags are like keywords which defines that how web browser will format and display the content. With the help of tags, a web browser can distinguish between an HTML content and a simple content.
HTML tags contain three main parts: opening tag, content and closing tag. But some HTML tags are unclosed tags.

There are Inline tags like heading tag, paragraph tag etc which needs an ending. There are Empty tags as well that does not need an ending like line break tag, horizontal rule tag, image tag etc.

h1 tag is heading tag which is used to make a heading.

h2, h3 and so on are used to make subheadings.

p tag is used to make paragraphs.

hr tag gives a horizontal rule to the heading.
